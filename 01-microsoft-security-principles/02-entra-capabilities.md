## Links

- [Entra ID](https://www.microsoft.com/en-us/security/business/microsoft-entra)

## Notes

- Previously known as Azure AD
- Cloud identity and access management solution
- Used By
  - M365 
  - Azure
  - Dynamics 
  - Etc..

- Made up of several products
  - Entra ID
    - Conditional Access
    - RBAC
    - Group Management
    - Session lifetime management
    - Sync with on-prem AD
    - Self Service password reset
  - Entra ID Protection
    - Detect and report identity based risks
  - Entra ID Governance
    - Review assigments and access reviews
  - Entra External ID
    - Allows external identities to access to apps and resources
  - Entra Domain services
    - Allows older apps to use cloud resources and auth services
  - Entra Workload ID
    - Control workload identity
