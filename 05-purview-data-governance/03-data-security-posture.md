# Microsoft Purview Data Security Posture Management for AI



## Links

- [Microsoft Learn](https://learn.microsoft.com/en-us/purview/ai-microsoft-purview)

## Notes

- Provides recommended policies for AI and a default risk assessment that is run every week for top 100 SharePoint sites based on usage

- Purview > Solutions > DSPM for AI

- Overview
  - Audit
  - Browser extension
  - Insights
    - Detect sensitive info shared in ai prompts in Edge
    - Detect when users visit AI sites
    - Detect when sensitive info is uploaded or pasted into ai sites
  - Recommendations
  - Reports
    - 24 hours for reports
  - Activity Explorer
  - Data Risk Assessment
