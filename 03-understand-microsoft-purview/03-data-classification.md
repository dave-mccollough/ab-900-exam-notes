## Notes

- Classification can happen three ways
- Manually
- Automated pattern matching
  - Keywords
  - Metadata
  - Document fingerprintind
  - Exact string/data match
- Trainable classifiers
  - Several pretrained classifiers
  - Can create custom classifiers
    - Positive sample set
      - 50-500 files
    - Negative sample set
      - 150-1500 documents
    - 2-12 days for training
