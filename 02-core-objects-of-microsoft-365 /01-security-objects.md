# Identify the appropiate security object to use in an organization

## Links

- [Microsoft Learn](https://learn.microsoft.com/en-us/entra/fundamentals/concept-learn-about-groups)

## Notes

- Users
  - Internal
    - Authenticates using your Microsoft tenant
    - Member
    - Guest
  - External
    - Authenticates using their Microsoft tenant
    - Member 
    - Guest
- Groups
  - Assign permissions to groups instead of individual users
  - Security
    - Manage access to applications, resources, etc
  - Microsoft 365
    - Collaboration
    - Teams, email account
  - Users added as assigned or dynamic
    - Dynamic: Users added to group based on criteria
  - Owners
    - Should be two owners
