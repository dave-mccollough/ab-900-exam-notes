# Identify the appropriate objects to configure by using the SharePoint in Microsoft 365 admin center (sites, libraries, and folders)

## Links

- [Microsoft Learn](https://learn.microsoft.com/en-us/sharepoint/manage-sites-in-new-admin-center)

## Notes

- SharePoint Admin Center
  - M365 Admin Center > Admin Centers > SharePoint
  - https://<company-name>-admin.sharpoint.com
  - SharePoint admin role or Global Admin

- Create Sites
  - Team Sites
  - Communication Sites

- Hubs
  - Connect sites together

- Membership
  - Owners
  - Members
    - Can edit site contents (files, etc)
  - Site Admins
    - Full control of site
  - Site Visitors
    - View only permissions
  
- Sharing permissions
