# Manage prompts, including Pages and Notebooks

## Links

- [Microsoft Learn](https://learn.microsoft.com/en-us/microsoft-365/loop/cpcn-storage?view=o365-worldwide)

## Notes

- Prompts should include    
  - Goal
  - Why
  - Context or meaning
  - Expectations or How
  - Source

- Pages
  - Click `Edit in Pages`
  - Interactive canvas for collaboration
  - You can edit and share
  - Can rename
  - Versioning
  - Open in Word
  - Control access
  - Add to Microsoft 365 app as a component

- Notebooks
  - Collaborative workspace for a task
  - Pull together all relevant resources
    - Microsoft 365 files
    - Copilot chats
    - Meeting notes
  - Versioning

- Editing Memory
  - Settings > Personalization > Copilot Memory

- Custom Instructions
  - Settings > Personalization > Custom Instructions
