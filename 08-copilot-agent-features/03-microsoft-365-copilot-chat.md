# Microsoft 365 Copilot and Copilot Chat

## Links

- [Microsoft Learn](https://learn.microsoft.com/en-us/copilot/overview)

## Notes

- Can enter prompts at office.com
- Can add attachments, images, etc
- To add work content
  - Click `+` icon in chat
  - Type `/` in chat


- If Copilot premium features are enabled
  - A Work and Web tab will appear at the top of the screen
    - If you select Web only web data will be used
    - If you select Work only work data will be used
