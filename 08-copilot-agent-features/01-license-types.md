# Copilot Licence Types

## Links

- [Microsoft Learn](https://learn.microsoft.com/en-us/microsoft-365/copilot/which-copilot-for-your-organization)

## Notes

- Copilot capabilities are dependent on plan selected
- Microsoft Copilot
  - Grounded in web and any files attached to the question
  - Grounded in current file
  - No license required
  - More features included in M365 Plan
  - https://copilot.microsoft.com

- Microsoft 365 Copilot
  - Grounded in web and any files attached to the question
  - Grounded in data that exists in SharePoint, etc
  - Need a Copilot business or enterprise plan

- Microsoft 365 Copilot Chat
  - Grounded in web and any files attached to the question
  - Not integrated into Microsoft apps
  - Functionality varies based on license
    - Not grounded in data that exists in SharePoint, etc
      - Need a Microsoft 365 license
    - Grounded in data that exists in SharePoint, etc
      - Need a Copilot business or enterprise plan
